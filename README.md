# The Roles I Balance – Project Website

## Overview
This project is a simple multi-page website created for COMP2300: Fundamentals of Computing. The purpose of this assignment is to demonstrate the use of core web technologies including HTML5, CSS, and JavaScript while following best practices such as Separation of Concerns and the Single Responsibility Principle.

The website focuses on a personal topic: balancing multiple roles in life, including work, education, and fatherhood.

---

## Project Structure
The website consists of four main pages:

- **index.html** – Main page introducing the topic and includes a 5-question quiz
- **page2.html** – Describes professional responsibilities and work experience
- **page3.html** – Covers educational background and career goals
- **page4.html** – Focuses on fatherhood and personal priorities
- **style.css** – Contains all styling for the website

---

## Features
- Multi-page navigation system
- Consistent styling across all pages using an external CSS file
- JavaScript-based quiz on the main page
- Embedded images
- External hyperlink for additional information
- Footer with copyright and contact information on every page

---

## Design Approach
This project follows the concept of **Separation of Concerns (SoC)**:
- HTML is used for structure and content
- CSS is used for styling and layout
- JavaScript is used for interactivity (quiz functionality)

This ensures the code is organized, maintainable, and easy to understand.

---

## Topic Description
The website explores the balance between three major roles:
- **Work** – Law enforcement and State Guard responsibilities
- **School** – Academic path in computing, GIS, and criminology
- **Family** – Responsibilities and priorities as a father

Each section provides a brief overview of the role and its importance.

---

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any modern web browser
